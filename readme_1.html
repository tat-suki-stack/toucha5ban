闘茶伍番勝負

札を伏せて茶を当てる、利き茶勝負。
の れ ん を く ぐ る（プレイする）

これは何？

鎌倉〜室町時代に武士や貴族が熱狂した利き茶賭博「闘茶」をモチーフにした、ブラウザで遊べる茶種当てゲームです。 伏せられた四枚の札——製法・水色・香味・産地——をめくって手がかりを集め、その茶が何かを見抜きます。 札を少なくめくって当てるほど高得点。全五服の勝負で、あなたの「茶の目利き」が試されます。

収録するのは日本茶14種を含む26種類の茶。遊ぶうちに、緑茶も烏龍茶も紅茶も、 すべて同じ一本の木——チャノキ（Camellia sinensis）——から生まれることが、体でわかる作りになっています。

遊び方

  1. のれんをくぐって入店する
  2. 「製・色・香・地」の四枚の札から、好きな札をタップしてめくる（最低一枚）
  3. 手がかりが揃ったと思ったら、四つの選択肢から茶を選ぶ
  4. 正解すると得点。めくった札が少ないほど高得点
  5. 五服すべて終えると、総点に応じた「位」が授けられる
めくった札一枚二枚三枚四枚
得点80604020

位は総点で決まります：茶聖（320点〜）／茶匠（240点〜）／茶人（160点〜）／見習い（80点〜）／客人。満点は400点です。

六大茶類のいろ

このゲームの配色は、中国茶の分類「六大茶類」——緑茶・白茶・黄茶・青茶（烏龍）・紅茶・黒茶——に由来します。 六つの色の違いは、実は発酵の度合いの違い。正解発表のたびに現れる発酵ゲージは、この六色の順にできています。

緑（不発酵）青（半発酵）紅〜黒（完全・後発酵）

つくり

  • 単一HTMLファイル——ビルド不要・サーバー不要・データベース不要。ダウンロードしてブラウザで開くだけで動きます
  • 外部依存はGoogle Fonts（Shippori Mincho / Zen Old Mincho / Yuji Boku / DM Mono）のみ。オフラインでもシステムの明朝体で動作します
  • 筑紫明朝・筑紫オールド明朝を最優先のフォント指定にしており、導入済みの環境ではそちらで表示されます
  • のれん・円相・カルタ札はすべてCSSとSVGによる描画で、画像ファイルを使っていません
  • スマートフォン（iOS Safari含む）対応。ホーム画面追加用アイコンとOGP設定済み
