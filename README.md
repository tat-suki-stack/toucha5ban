# 闘茶伍番勝負 — TOUCHA: FIVE ROUNDS

札を伏せて茶を当てる、利き茶勝負。

鎌倉〜室町時代に武士が熱狂した利き茶賭博「闘茶」をモチーフにした、ブラウザで遊べる茶種当てゲームです。「製法・水色・香味・産地」の四枚の札をめくって手がかりを集め、その茶が何かを見抜きます。札を少なくめくって当てるほど高得点。全五服。

## 特徴

- 日本茶14種を含む全26種の茶を収録（煎茶・玉露から碁石茶・阿波晩茶まで）
- 配色は中国茶の分類「六大茶類」（緑・白・黄・青・紅・黒）に由来。発酵ゲージで「すべて同じチャノキ」が体感できる
- のれんをくぐって入店するスプラッシュ演出、筆の円相、カルタ調の縦書き札
- 単一HTMLファイル。ビルド不要・サーバー不要・DB不要（外部依存はGoogle Fontsのみ）
- スマートフォン対応。ホーム画面追加用アイコン・OGP設定済み

## 遊び方

1. のれんをくぐって入店
2. 四枚の札から好きな札をめくる（最低一枚）
3. 四択から茶を選ぶ — めくった札が少ないほど高得点（一枚80点〜四枚20点）
4. 五服の総点で「位」が決まる（茶聖／茶匠／茶人／見習い／客人・満点400点）
